Steps to use this project on local device:

1) Clone this repository

2) while being in the project root directory use command >npm install

3) Then >npm start

4) Visit this link before starting local frontend 
https://kevinujunior-live-leaderboard-backend.azurewebsites.net/

4) go to http://localhost:3000/ in your browser and enjoy
